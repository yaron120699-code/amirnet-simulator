# PrepLab — v0.9.7 English Level Beta

גרסה זו מפשטת את שכבת פירוש הציון: בלי מוסדות לימוד ובלי ספי שיבוץ לפי אוניברסיטה/מכללה.
המטרה היא להציג למשתמש **רמת אנגלית משוערת** בלבד: רמת התחלה, טרום בסיסי, בסיסי, מתקדמים א׳, מתקדמים ב׳ או פטור.

## מה חדש

- הוסר בורר מוסד הלימודים ממסך הבית
- הוסר `data/institutions.js`
- נוסף `data/levels.js` עם טווחי רמות כלליים בלבד
- `js/placement.js` פושט: אין מוסדות, רק סיווג רמת אנגלית
- במסך התוצאות מופיעים:
  - ציון אמירנט משוער
  - רמת אנגלית משוערת
  - היעד הבא: פטור
  - כמה נקודות חסרות לפטור
  - פס ויזואלי 50→150
  - משפט מוטיבציה קצר לפי הרמה
- המנוע האדפטיבי, הציון, השאלות וה־Blueprint לא שונו

## הערה חשובה

רמת האנגלית היא הערכה בלבד על בסיס הסימולציה. הרמה הרשמית נקבעת רק על ידי מבחן אמירנט הרשמי.
אין כרגע תמיכה במוסדות לימוד ספציפיים בכוונה — זה יישאר פשוט ואמין יותר לגרסת הבטא.
