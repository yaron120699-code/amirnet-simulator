# PrepLab — v0.9.8 Vocabulary Engine

גרסה זו מוסיפה שכבת אוצר מילים פנימית ל־PrepLab, בלי לשנות את חוויית המשתמש הציבורית.

## מה חדש

- `data/vocabularyBank.js` — מאגר ראשוני של 800 מילים מתוך הרשימה שסופקה.
- `js/vocabularyEngine.js` — מנוע פנימי לחיפוש מילים, סטטיסטיקות ויצירת טיוטות שאלות.
- `vocab-studio.html` — כלי פנימי נסתר ליצירת טיוטות JSON ממילים.

## חשוב

`vocab-studio.html` לא מקושר מהאתר הציבורי. נכנסים אליו ידנית רק אם יודעים את הכתובת:

```text
/vocab-studio.html
```

זהו כלי פנימי בלבד. השאלות שהוא יוצר הן טיוטות ודורשות בדיקה אנושית לפני כניסה ל־`questionBank.js`.

## מה לא השתנה

- המנוע האדפטיבי לא שונה.
- מנגנון הציון לא שונה.
- בנק השאלות הציבורי לא שונה.
- מסך הבית לא מציג קישור ל־Vocabulary Studio.

## קבצים להעלות

להעלות את כל הקבצים והתיקיות לשורש הריפו:

```text
index.html
css/
js/
data/
questions/
docs/
vocab-studio.html
README.md
```

- [ ] החיפוש עובד לפי מילה, תרגום ונושא.
- [ ] בחירת מילה מייצרת JSON.
- [ ] כפתור העתקה עובד.

## הערת איכות

המאגר הראשוני נוצר אוטומטית מהרשימה שסופקה. רמות הקושי, חלקי הדיבר והנושאים הם הערכה ראשונית ודורשים כיול עם הזמן.
