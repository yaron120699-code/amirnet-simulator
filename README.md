# Amirnet Sim v0.2

גרסת סימולציות אנגלית לאמירנט.

## איך מפעילים
1. חלץ את קובץ ה-ZIP.
2. פתח את `index.html` בדפדפן.

## מה חדש ב-v0.2
- מאגר שאלות גדל ל-120 שאלות מקוריות.
- חלוקה מאוזנת: 30 Sentence Completion, 30 Restatement, 30 Vocabulary in Context, 30 Reading Comprehension.
- 3 רמות קושי לכל סוג שאלה.
- מצב ללא זמן נשאר פעיל.
- ציון משוער 50–150 לפי קושי השאלות.

## קבצים
- `index.html` — פתיחת האפליקציה.
- `app.js` — מנוע המבחן, טיימר, תוצאות.
- `questions.js` — בנק השאלות.
- `styles.css` — עיצוב.

הציון הוא הערכה פנימית לצורך תרגול ולא ציון רשמי.
