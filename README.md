# Amirnet Simulator v0.3

גרסה ראשונית עם מנגנון אדפטיבי:

- מתחיל ברמת יכולת 3 מתוך 5
- תשובה נכונה מעלה את הרמה
- תשובה לא נכונה מורידה את הרמה
- השאלה הבאה נבחרת לפי הרמה הנוכחית
- בסוף מתקבל ציון משוער בטווח 50–150
- הציון משקלל דיוק, קושי, רמת יכולת ממוצעת ורמת יכולת סופית

חשוב: זה מנגנון פנימי שמחקה את רעיון האדפטיביות. זה לא האלגוריתם הרשמי של אמירנט.
